Welcome to Social Media API

This is a RESTful API for a social media platform. Use the following endpoints to interact:

  • POST /api/register/: Create a new user account and get a token.
  • POST /api/login/: Log in and retrieve a token.
  • GET /api/profile/: View your user profile (requires token).

Test the API using tools like Postman. Admin panel available at /admin/.
